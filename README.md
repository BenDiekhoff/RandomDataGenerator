# RandomDataGenerator
Generates random user data in a csv or json file.  
Run ```dataGenDiekhoff.py <file type> <number of users to generate>```  
from the command line, where `<file type>` is either 'csv' or 'json' 

# Data
- user_id
- email
- username
- first_name
- last_name
- password
- create_time
- last_update
